# Plato Agency Frontend Readme

This folder contains module-level documentation for the frontend. Each module
README includes:
- Purpose and responsibilities
- Key components and stores
- Primary routes and API interactions

## Module Index
- `auth.md` - login, signup, reset, verify flows
- `dashboard.md` - shell layout, sidebar, navbar
- `jobs.md` - jobs list, job details, job watch, AI prompt
- `resumes.md` - resume list, details, actions
- `ui.md` - shared UI components and patterns

## Shared Notes
- Routing uses Next.js app router in `app/`.
- API calls live in stores under `shared/store`.
- Toast messages use `shared/helper/toast`.
