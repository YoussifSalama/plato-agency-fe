# UI Module

## Purpose
Reusable UI components and patterns shared across pages.

## Key Components
- `components/ui/button.tsx`
- `components/ui/dropdown-menu.tsx`
- `components/ui/table.tsx`
- `components/ui/tooltip.tsx`
- `components/ui/input.tsx`
- `components/ui/textarea.tsx`
- `components/ui/select.tsx`

## Notes
- Styling is Tailwind-based with shared variants.
- Tooltip uses a portal to avoid overflow clipping.
